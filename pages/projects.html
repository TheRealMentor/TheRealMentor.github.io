Reconnaissance Bot

Mar 2018 ‐ Apr 2018

This bot is made for recon. It uses a camera, attached in front, for object identification and tracking. It is built for difficult rough terrains with elevations and gravel, where normal bots can't reach. It is remotely controlled through RF module, so no worries about the range of the bot. It uses sensors to detect any moving object nearby and move the camera towards the moving object.
